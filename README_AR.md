# Karkariya-Noor / الكركريه نور

مشروع Android Studio أولي جاهز للبناء، مبني حسب التصميم المرجعي المرسل: ألوان كريمية، أخضر داكن، ذهبي، صورة الشيخ، وأيقونة التطبيق.

## الموجود في هذه النسخة
- Splash Screen بالأيقونة.
- الصفحة الرئيسية.
- صفحة الشيخ مع السيرة والسلسلة الذهبية.
- صفحة أصول الطريقة بالأصول السبعة.
- ورد اليوم مقسم إلى بطاقات.
- سبحة ذكية مرسومة Canvas مع صوت واهتزاز وصيغة إكمال.
- مكتبة offline بسيطة قابلة للتوسعة.
- قسم المرئيات والأخبار يفتح المواقع الرسمية ويحتاج إنترنت.
- التواصل مع المكلفين: مصر والمغرب عبر واتساب.
- إعدادات اللغة والثيم والتنبيهات.
- المحتوى مخزن في assets/content/ar و assets/content/en لتعديله بسهولة.

## البناء
افتح المجلد في Android Studio ثم اضغط Sync وبعدها Build APK.

## ملاحظات مهمة
- لم أستطع بناء APK داخل هذه البيئة لأنها لا تحتوي Android SDK/Gradle.
- راجع النصوص الشرعية والسلسلة والأوراد قبل النشر العام.
- يمكنك استبدال الصور من res/drawable-nodpi مع الحفاظ على نفس الأسماء.
